# pickr

Pick randomly

![Go](https://github.com/thealamu/pickr/workflows/Go/badge.svg)

## Install

```
$ go get github.com/thealamu/pickr/cmd/pickr
```

## Usage

### Toss a coin

```
$ pickr toss
Heads
```

### Roll a 8-sided die

```
$ pickr roll 8
3
```
